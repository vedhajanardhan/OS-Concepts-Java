# OS-Concepts-Java.

✨ From theory to code — bringing Operating System fundamentals to life with Java.  
   This repo explores core OS concepts through hands-on implementations, designed to strengthen CS fundamentals and showcase practical
   problem-solving.
   
## 📚 Concepts Covered
- ⚡ CPU Scheduling → Round Robin implementation.
- 🔒 Deadlock Avoidance → Banker’s Algorithm.
- 🔄 Process Synchronization → Producer-Consumer Problem.

## 🚀 How to Run
1. Clone the repo
2. Navigate to the desired folder (`Scheduling`, `Deadlocks`, `Synchronization`)
3. Compile using `javac FileName.java`
4. Run using `java FileName`

## 📂 Structure
- `/Scheduling` → Round Robin Scheduling
- `/Deadlocks` → Banker’s Algorithm
- `/Synchronization` → Producer-Consumer Problem

## 🎯 Purpose
A creative collection of mini-projects built to:
- Reinforce OS fundamentals
- Prepare for SDE-1 interviews
- Showcase applied knowledge in Java
